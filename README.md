# Heart Disease Prediction using Machine Learning

## Overview

This project was developed as the Week 4 Major Capstone Project for the Skill Nexis Machine Learning & AI Internship.

The objective of this project is to predict whether a patient is likely to have heart disease based on medical attributes using Machine Learning techniques.

## Dataset

- Dataset: Heart Disease UCI Dataset
- Source: Kaggle / UCI Repository
- Records: 920
- Features: 22 (after preprocessing)

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Missing Value Handling
4. Feature Encoding
5. Feature Scaling
6. Train-Test Split
7. Logistic Regression Model
8. Decision Tree Model
9. Model Evaluation
10. ROC Curve Analysis
11. Feature Importance Analysis
12. Model Saving and Loading

## Models Implemented

### Logistic Regression

- Accuracy: 86.96%
- Precision: 86.79%
- Recall: 90.20%
- F1 Score: 88.46%

### Decision Tree

- Accuracy: 82.61%
- Precision: 84.31%
- Recall: 84.31%
- F1 Score: 84.31%

## Best Model

Logistic Regression achieved the highest performance and was selected as the final model.
